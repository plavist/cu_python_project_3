Для визуализации погодных данных в Python с Plotly лучше всего подходят линейные графики (для трендов температуры, влажности и т.д. по времени), 
столбчатые диаграммы (для количества осадков), графики рассеяния (для корреляции между параметрами), 
графики областей (для накопленных значений) и карты (для географического распределения); 
улучшение пользовательского опыта достигается за счет интерактивности: наведение курсора для деталей, выбор диапазона для фокусировки на периодах, 
масштабирование/панорамирование для детального изучения, управление легендой для сравнения параметров и экспорт для сохранения/обмена графиками.
